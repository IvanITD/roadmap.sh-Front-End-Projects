# Single Page CV

A single-page HTML resume built as part of the [roadmap.sh](https://roadmap.sh/projects/single-page-cv) front-end curriculum. Focused on semantic HTML, accessible markup, and clean code structure.

## Features

- Semantic HTML5 elements (`header`, `main`, `section`, `footer`)
- Accessible markup with proper heading hierarchy
- Clean, professional layout
- Personal details stored separately for privacy

## Technologies

- HTML5
